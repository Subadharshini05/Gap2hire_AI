# Gap2Hire – UI Design Plan

## Color Palette
- Black: Primary background
- White: Text and content cards
- Red: CTA buttons, highlights, skill gaps

## Design Style
- Minimal
- High contrast
- Professional HR-tech look
- No animations or gradients

## Screens
1. Landing Screen
   - Resume input
   - Job description input
   - Analyze Resume button

2. Analysis Screen
   - Resume-job match score
   - Missing skills
   - Resume feedback
   - Practice Interview button

3. Interview Prep Screen
   - Skill-based interview questions
   - Simple layout

## Design Goal
Clear and explainable flow:
Input → Analysis → Interview Practice
